# Minesweeper Game 🧨

A simple browser-based Minesweeper game built using **HTML**, **CSS**, and **JavaScript**.

## 🔹 Features
- Click to reveal cells
- Right-click to flag mines
- Recursive reveal for empty cells
- Win/loss logic

## 🚀 Tech Stack
- HTML
- CSS
- JavaScript
